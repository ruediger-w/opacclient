Veškerá komunikace probíhá mezi vaším chytrým telefonem a serverem knihovny. Provoz služby Web Opac nevyžaduje ukládání dat, nedochází tedy ani k jejich ukládání. Na vašem zařízení jsou ukládány výsledky hledání. Provozovatel služby nemá informace o tom, zdali-li a případně jaké informace ukládá server vaší knihovny. Pokud vás zajímají podrobnosti, obraťte se na její pracovníky.

In regular intervals, the app contacts our server to fetch updated data about the libraries. Also, the app may contact our server in case of an application error. We need this to ensure the ongoing quality of the app, but we will never transmit or store personal data.

Only exception: When you share a search result from the app to your friends, the link that your friend saves takes a detour through our server (if he does not have the app installed and set-up for the same library). In this case, our server will save the time of the request and which book was requested, but no IP addresses, browser data etc. This is necessary because this "Sharing" feature is something that most library systems do not support natively, so we need to rebuild it using this technique.
